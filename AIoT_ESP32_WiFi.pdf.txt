1. 功能介紹 
此小節將介紹 ESP32 的 WIFI 運作。 
ESP32 的 WIFI 運作模式 
ESP32 的 WIFI 運作模式可以分為三種 :  
 
1.AP (Access Point) 模式  
  允許其他無線設備接入，就像是住家或公共區域的無線網路基地台。 
 
2.STA (Station) 模式 
  不允許其他無線設備接入，只能去連接其他 AP 的裝置。 
 
3.AP/STA 混和模式 
  同時當作基地台，也可以去連接其他裝置。 
  
在接下來的小節，我們會設定 ESP32 為 STA 模式，並連接手機網路，將資料上傳
至網頁或是資料庫中。 
 
  
2. 掃描附近 WIFI 
此小節將介紹如何使用 ESP32 去掃描附近的 WIFI。 
1. 掃描 WIFI 的程式碼 
程式碼如下 
#include "WiFi.h"        //WIFI 的標頭檔 
 
void setup() { 
  Serial.begin(115200); 
  WiFi.mode(WIFI_STA);   //設定 ESP32 的網路模式為STA 模式 
  WiFi.disconnect();     //將先前 WIFI 的連結中斷掉(初始化) 
  delay(100); 
  Serial.println("Setup done"); 
} 
 
void loop() { 
  Serial.println("scan start"); 
  int n = WiFi.scanNetworks();   //掃描附近網路 
  Serial.println("scan done"); 
 
  if (n == 0) 
    Serial.println("no networks found");
